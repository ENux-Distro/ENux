# Calamares ENux Installer

Calamares ENux Installer, is an GUI installer for ENux, that is a customized *Calamares Debian Installer*. 
Calamares ENux Installer, is where Bedrock Linux get's installed via the **--hijack** operation.

### How to Install Calamares ENux Installer

In order to install Calamares ENux Installer, do these:

- Grab the .deb file from [here](https://github.com/ENux-Distro/ENux/raw/refs/heads/main/calamares-settings-enux.deb)
- install it via **dpkg --install *** /path/to/*calamares-settings-enux**
- Enjoy
