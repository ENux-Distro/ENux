# ENux Finalizer

ENux Finalizer is a tool for ENux, that **brl fetches**

- Alpine
- Void
- Arch
- CentOS
- Gentoo
- openSUSE

and installs **nix** as an extra toolchain

### How to Install ENux Finalizer

- Grab the ENux Finalizer script from [here](https://github.com/ENux-Distro/ENux/raw/refs/heads/main/enux-finalizer)
- Run the Finalizer with **sudo bash** **/path/to/enux-finalizer**
- Enjoy
