# ENux Welcomer

ENux Welcomer is a tool for ENux, that

- Installs recommended packages
- Explains what ENux is
- **brl fetches**
  - Alpine
  - Void
  - Arch
  - CentOS
  - Gentoo
  - openSUSE
  - installs **nix** as an extra toolchain
- Let's you test a package manager (Void Linux's xbps)

### How to Install ENux Finalizer

- Grab the ENux Finalizer script from [here](https://github.com/ENux-Distro/ENux/raw/refs/heads/main/enux-welcomer)
- Run the Finalizer with **sudo bash** **/path/to/enux-finalizer**
- Enjoy
