WHAT IS ENUX 1.0?

ENux 1.0 is a Linux Distribution which is based on Debian and uses Bedrock Linux on top to be able to use
        
        -apk
        
        -xbps-install
        
        -dnf
        
        -rpm
        
        -emerge
        
        -pacman
        
        -dpkg
        
        -apt
        
        -enux
But you may ask, how did we fix conflicts? Bedrock Linux fixes most of the headaches, but if you are a beginner, use enux, which is our pmm, making things easier for beginners

HOW TO INSTALL ENUX 1.0?
Grab the ENux 1.0 ISO from "www.emirpasha.com" or from "https://www.emirpasha.com/ENux-1.0.iso"

After downloading the ISO, flash it to an USB using tools like Rufus or Belena Etcher.

Then boot from the USB that you just flashed the ISO at, and then run ENux Installer.

After you reboot, immediately run ENux Finaliser Phase 1, which installs Bedrock Linux, and neceserry tools. 

After the script reboots your machine, immediately run ENux Finaliser Phase 2, which installs the package managers (by brl fetching them).
